# Cider 2 — the Listening Room's deliberate exception

Cider 2 (paid, itch.io / Taproom) is the Apple Music lane; rmpc + cava stay
the local-library centerpiece. The reason both feel native to the room:
**MPRIS is the spine.** Cider speaks it natively, mpd-mpris bridges MPD onto
the same bus, so the waybar now-playing widget, playerctl media keys, and
the room's controls treat them identically.

Dress it: paste `enfilade-amber.css` into Settings → Visual → custom
style/theme editor. The window is already ruled into room 4 silently.
