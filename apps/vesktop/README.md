# Vesktop + system24 — Discord in a terminal costume

install.sh fetched refact0r's system24 into `~/.config/vesktop/themes/` and
copied `enfilade-cables.theme.css` (a CABLES-palette flavor) beside it.

Enable: Vesktop → Settings → Vencord → Themes → toggle **enfilade-cables**
(it @imports the hosted system24 build, so it works alone). The window is
already window-ruled into room 2.

Voice/video lives here — that's this lane's whole point versus endcord.
